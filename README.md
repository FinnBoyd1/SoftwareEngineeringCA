# SoftwareEngineeringCA

Website application designed to provide users with an all-in-one platform that will let
them search tv shows or films that have either just released or will be released.​

The website will allow users to quickly search, filter, and explore content using reliable
information from other media platforms, all within a clean and accessible interface.​


Programming Languages Used: HTML, CSS & Tailwind CSS, JavaScript, NodeJS​

API from TMDB​

MongoDB Atlas is used for the DB
